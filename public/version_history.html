Swimbot Speak Version History

Version 0.3 (07 May 2023)

Description:

- Response is now returned as a JSON string which is parsed into a response and current action the swimbot is taking.
- The current action is displayed below the response

Version 0.2 (01 May 2023)

Description:

- Added controls for tweaking the swimbot's brain, including personality and mental state.

Version 0.1 (30 April 2023)

Description:

Initial release of the Swimbot Speak engine that allows users to converse with a swimbot. The swimbot has a memory of 8 messages and 8 replies.
